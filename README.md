
###      Premium Digital Tools Platform    ####



## Project Description
This is a modern and responsive web application where users can explore premium digital tools, view pricing plans, and add products to a cart. The platform provides a clean UI interaction, making it easy for users to browse and manage tools easily.



##  Technologies Used
 React.js
 Tailwind CSS
 DaisyUI
 Vite
 React Toastify



##  Features
 1.  Add to Cart System
Users can easily add tools to the cart with real-time updates.
2.  Tab-based Navigation
Smooth switching between 'Products' and 'Cart' sections.
 3. Pricing Plan Section
Displays different subscription plans.
